============
Installation
============

.. contents::
   :local:

You can install hop-plugin-{{ cookiecutter.plugin_name}} via pip or from source.

To install with pip:

.. code:: bash

   pip install -U hop-plugin-{{ cookiecutter.plugin_name}}

To install from source:

.. code:: bash

    tar -xzf hop-plugin-{{ cookiecutter.plugin_name}}-x.y.z.tar.gz
    cd hop-plugin-{{ cookiecutter.plugin_name}}-x.y.z
    python setup.py install
